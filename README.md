# Финансовый журнал для студентов

## Описание
Приложение для учёта доходов и расходов студентами с просмотром баланса и формированием отчётов.

## Цели
- Учёт доходов и расходов  
- Планирование бюджета  
- Отчёты и статистика  

## Технологии
Figma, Git/GitHub, Excel/SQLite, Flutter/React Native/Web

## Модули
- UI: регистрация, вход, главная панель, отчёты  
- Логика: обработка данных, пересчёт баланса, графики  
- Данные: хранение и структура  
- Вспомогательные: уведомления, настройки, интеграции  

## Основные требования
- Ввод доходов/расходов  
- Авторасчёт баланса  
- Статистика и отчёты  
- Редактирование и удаление записей
